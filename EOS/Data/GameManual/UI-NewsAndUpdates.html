News and Updates

The "News and Updates" interface connects to a webserver, gives users news involving new software versions, allows users to download new maps, scenarios, rulesets, and versions of the application.


News

The News tab gives users the latest information on the game, including server downtimes and application updates.


Updates

The Updates tab tells users about updates to files (application, maps, scenarios, and rulesets) that they have previously downloaded.

Items marked with a blue circling arrow are files that have been updated since you downloaded it. Clicking on the item allows you to download the latest version of the file.

Items marked with a download symbol are items that have been updated.


Download

The Download tab allows users to download new maps, scenarios, and rulesets.

Items marked with a small tic have not been downloaded. Clicking on this item will download it.

Items marked with a download symbol are items that have been downloaded.


Share

The Share tab allows users to upload maps, scenarios, and rulesets that they have created. Users can also check on the scores and comments that other users have given.

Items marked with a small tic have not been shared. Clicking on this item will upload the file so other users can download it.

Items marked with a share symbol are items that have been shared. Clicking on the item allows you to stop sharing the file.

Items marked with a unshared symbol are items that were shared, but have been unshared. Clicking on the item allows you to resume sharing the file.

Items marked with a blue circling arrow are files that you've uploaded, but changed since the upload. They need to be re-uploaded so other players can get the latest version. Clicking on the item allows you to upload the file or stop sharing the file. (When you re-upload the file, the file will appear in other user's "Updates" list if they have previously downloaded the file.)

Create Account

Users can create an account. An account is required to upload files, vote, or comment on other people's files.

Sign-in

To access your account, you must be signed-in. The application can automatically remember your password, so you don't need to type it in each time.
